# Installation

This document records every installation step performed during server deployment.

## Operating System

Ubuntu 25.04 (Plucky Puffin)

## Initial Tasks Completed

- Fresh Ubuntu installation
- System package index updated
- Reviewed available package upgrades
- Created a Proxmox snapshot (`before-first-update`)
- Upgraded installed packages
- Performed post-upgrade verification

## Notes

The upgrade was performed only after taking a VM snapshot to provide a rollback point in case of unexpected issues.
