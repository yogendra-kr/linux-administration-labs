# Ubuntu Server Commissioning

## Project Goal

This project documents the process of commissioning and hardening an Ubuntu Server from a fresh installation.

The goal is to follow professional Linux system administration practices while documenting every decision, configuration change, and verification step.

## Project Objectives

- Build a secure Ubuntu server
- Follow Linux administration best practices
- Document every configuration change
- Automate repetitive administrative tasks
- Practice production-style troubleshooting

## Technologies

- Ubuntu Server
- OpenSSH
- UFW
- Bash
- Proxmox VE
